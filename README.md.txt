# ClickCounter4

## Description
ClickCounter4 is a simple, non-functional click counter app developed as a proof of concept to demonstrate that individuals with no prior programming experience can successfully create functional applications using AI (ChatGPT) as a guide. The app was built using Android Studio Jellyfish with Kotlin and Jetpack Compose.

## Requirements
- Android Studio Jellyfish 2021.3.1 or later
- Kotlin 1.5.x (compatible with Jetpack Compose 1.5.1)
- Jetpack Compose 1.5.1
- Compile SDK Version 34
- Minimum SDK Version 26
- Target SDK Version 34
- Java 1.8 Compatibility

## Installation
1. Clone the repository:

git clone https://github.com/hollida4/Clickcounter4

2. Open the project in Android Studio.
3. Build and run the application on an emulator or physical device.

## Usage
Open the app to see a simple click counter interface. This version is intended to show the basic setup of a Kotlin project in Android Studio.
